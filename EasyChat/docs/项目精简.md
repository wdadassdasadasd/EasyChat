# 项目经历

## EasyChat 桌面即时通讯客户端

**技术栈：** Electron、Vue 3、Element Plus、Pinia、SQLite、WebSocket、Vitest

**简介：** 基于 Electron + Vue 3 开发的桌面端即时通讯客户端，支持登录、会话列表、单聊/群聊、联系人管理、文本/图片/视频/文件消息收发、本地历史记录与实时消息同步。

1. 搭建 Electron 主进程、Preload、Vue 渲染层三层架构，将 SQLite、WebSocket、窗口控制与 IPC 通信集中在主进程处理，提升客户端数据链路的可维护性。
2. 实现聊天消息收发链路，支持本地 pending 消息、服务端 messageId 替换、失败重试、媒体文件异步上传与上传并发控制，优化弱网场景下的消息状态体验。
3. 设计 WebSocket 心跳保活、断线重连、消息合法性校验、接收队列与批量落库机制，降低高频消息下的数据库写入和页面刷新压力。
4. 基于 SQLite 实现本地消息缓存、分页加载、消息搜索、会话清空游标、未读数维护与消息去重，保证消息表和会话表数据一致性。
5. 针对长聊天记录实现虚拟列表，基于消息高度缓存、前缀和与二分查找计算可视区域，仅渲染视口附近消息，提升历史消息滚动性能。
