# 后端数据库设计

来源：`D:\Easychat\easychat后端\easychat.sql`

数据库：MySQL，字符集 `utf8mb4`。本文档按后端 SQL 的表结构和字段 `COMMENT` 整理，方便后续前后端联调和聊天功能开发。

## 表总览

| 表名 | 表说明 |
| --- | --- |
| `app_update` | app 发布/更新信息 |
| `chat_message` | 聊天消息表 |
| `chat_session` | 会话信息表 |
| `chat_session_user` | 会话用户关系表 |
| `group_info` | 群组信息表 |
| `user_contact` | 用户联系人/群组关系表 |
| `user_contact_apply` | 联系人申请表 |
| `user_info` | 用户信息表 |
| `user_info_beauty` | 靓号表 |

## app_update

表说明：app 发布/更新信息。

| 字段名 | 类型 | 约束/默认值 | 字段含义 |
| --- | --- | --- | --- |
| `id` | `int(11)` | `NOT NULL AUTO_INCREMENT` | 自增 ID |
| `version` | `varchar(10)` | `NULL DEFAULT NULL`，唯一索引 `idx_key` | 版本号 |
| `update_desc` | `varchar(500)` | `NULL DEFAULT NULL` | 更新描述 |
| `create_time` | `datetime` | `NULL DEFAULT NULL` | 创建时间 |
| `status` | `tinyint(1)` | `NULL DEFAULT NULL` | 发布状态：`0` 未发布，`1` 灰度发布，`2` 全网发布 |
| `grayscale_uid` | `varchar(1000)` | `NULL DEFAULT NULL` | 灰度发布用户 ID 列表 |
| `file_type` | `tinyint(1)` | `NULL DEFAULT NULL` | 文件类型：`0` 本地文件，`1` 外链 |
| `outer_link` | `varchar(200)` | `NULL DEFAULT NULL` | 外链地址 |

主键：`id`

索引：`idx_key(version)` 唯一索引

## chat_message

表说明：聊天消息表。发送文本、文件等消息后，后端会在这里保存消息记录。

| 字段名 | 类型 | 约束/默认值 | 字段含义 |
| --- | --- | --- | --- |
| `message_id` | `bigint(20)` | `NOT NULL AUTO_INCREMENT` | 消息自增 ID |
| `session_id` | `varchar(32)` | `NOT NULL`，索引 `idx_session_id` | 会话 ID，用于归属到某个聊天会话 |
| `message_type` | `tinyint(1)` | `NOT NULL` | 消息类型。具体枚举需以后端代码为准 |
| `message_content` | `varchar(500)` | `NULL DEFAULT NULL` | 消息内容，文本消息主要使用该字段 |
| `send_user_id` | `varchar(12)` | `NULL DEFAULT NULL`，索引 `idx_send_user_id` | 发送人 ID |
| `send_user_nick_name` | `varchar(20)` | `NULL DEFAULT NULL` | 发送人昵称 |
| `send_time` | `bigint(20)` | `NULL DEFAULT NULL`，索引 `idx_send_time` | 发送时间，通常为毫秒时间戳 |
| `contact_id` | `varchar(12)` | `NOT NULL`，索引 `idx_receive_contact_id` | 接收联系人 ID。单聊时通常是接收用户 ID，群聊时通常是群 ID |
| `contact_type` | `tinyint(1)` | `NULL DEFAULT NULL` | 联系人类型：`0` 单聊，`1` 群聊 |
| `file_size` | `bigint(20)` | `NULL DEFAULT NULL` | 文件大小，文件消息使用 |
| `file_name` | `varchar(200)` | `NULL DEFAULT NULL` | 文件名，文件消息使用 |
| `file_type` | `tinyint(1)` | `NULL DEFAULT NULL` | 文件类型，具体枚举需以后端代码为准 |
| `status` | `tinyint(1)` | `NULL DEFAULT 1` | 消息状态：`0` 正在发送，`1` 已发送 |

主键：`message_id`

索引：`idx_session_id(session_id)`、`idx_send_user_id(send_user_id)`、`idx_receive_contact_id(contact_id)`、`idx_send_time(send_time)`

与前端聊天页关系：

| 前端用途 | 后端字段 |
| --- | --- |
| 展示消息内容 | `message_content` |
| 判断是否自己发送 | `send_user_id` |
| 区分单聊/群聊 | `contact_type` |
| 当前聊天窗口消息归属 | `session_id` |
| 消息排序 | `message_id` 或 `send_time` |

## chat_session

表说明：会话信息表。保存一个会话的摘要信息。

| 字段名 | 类型 | 约束/默认值 | 字段含义 |
| --- | --- | --- | --- |
| `session_id` | `varchar(32)` | `NOT NULL` | 会话 ID |
| `last_message` | `varchar(500)` | `NULL DEFAULT NULL` | 最后接收的消息/最后一条消息摘要 |
| `last_receive_time` | `bigint(11)` | `NULL DEFAULT NULL` | 最后接收消息时间，毫秒时间戳 |

主键：`session_id`

## chat_session_user

表说明：会话用户关系表。用于表示某个用户和某个联系人/群之间对应哪个会话。

| 字段名 | 类型 | 约束/默认值 | 字段含义 |
| --- | --- | --- | --- |
| `user_id` | `varchar(12)` | `NOT NULL`，联合主键 | 用户 ID |
| `contact_id` | `varchar(12)` | `NOT NULL`，联合主键 | 联系人 ID。单聊时是对方用户 ID，群聊时是群 ID |
| `session_id` | `varchar(32)` | `NOT NULL`，索引 `idx_session_id` | 会话 ID |
| `contact_name` | `varchar(20)` | `NULL DEFAULT NULL` | 联系人名称。单聊是联系人昵称，群聊是群名称 |

主键：`user_id, contact_id`

索引：`idx_user_id(user_id)`、`idx_session_id(session_id)`

## group_info

表说明：群组信息表。

| 字段名 | 类型 | 约束/默认值 | 字段含义 |
| --- | --- | --- | --- |
| `group_id` | `varchar(12)` | `NOT NULL` | 群 ID |
| `group_name` | `varchar(20)` | `NULL DEFAULT NULL` | 群组名 |
| `group_owner_id` | `varchar(12)` | `NULL DEFAULT NULL` | 群主 ID |
| `create_time` | `datetime` | `NULL DEFAULT NULL` | 创建时间 |
| `group_notice` | `varchar(500)` | `NULL DEFAULT NULL` | 群公告 |
| `join_type` | `tinyint(1)` | `NULL DEFAULT NULL` | 加群方式：`0` 直接加入，`1` 管理员同意后加入 |
| `status` | `tinyint(1)` | `NULL DEFAULT 1` | 群状态：`1` 正常，`0` 解散 |

主键：`group_id`

## user_contact

表说明：用户联系人/群组关系表。好友关系和群成员关系都在这张表中。

| 字段名 | 类型 | 约束/默认值 | 字段含义 |
| --- | --- | --- | --- |
| `user_id` | `varchar(12)` | `NOT NULL`，联合主键 | 用户 ID |
| `contact_id` | `varchar(12)` | `NOT NULL`，联合主键，索引 `idx_contact_id` | 联系人 ID 或群组 ID |
| `contact_type` | `tinyint(1)` | `NULL DEFAULT NULL` | 联系人类型：`0` 好友，`1` 群组 |
| `create_time` | `datetime` | `NULL DEFAULT NULL` | 创建时间 |
| `status` | `tinyint(1)` | `NULL DEFAULT NULL` | 关系状态：`0` 非好友，`1` 好友，`2` 已删除好友，`3` 被好友删除，`4` 已拉黑好友，`5` 被好友拉黑 |
| `last_update_time` | `timestamp` | `NULL DEFAULT NULL ON UPDATE CURRENT_TIMESTAMP` | 最后更新时间 |

主键：`user_id, contact_id`

索引：`idx_contact_id(contact_id)`

## user_contact_apply

表说明：联系人申请表。好友申请和加群申请都在这张表中。

| 字段名 | 类型 | 约束/默认值 | 字段含义 |
| --- | --- | --- | --- |
| `apply_id` | `int(11)` | `NOT NULL AUTO_INCREMENT` | 自增 ID |
| `apply_user_id` | `varchar(12)` | `NOT NULL`，唯一索引 `idx_key` 组成字段 | 申请人 ID |
| `receive_user_id` | `varchar(12)` | `NOT NULL`，唯一索引 `idx_key` 组成字段 | 接收人 ID |
| `contact_type` | `tinyint(1)` | `NOT NULL` | 联系人类型：`0` 好友，`1` 群组 |
| `contact_id` | `varchar(12)` | `NULL DEFAULT NULL`，唯一索引 `idx_key` 组成字段 | 联系人/群组 ID |
| `last_apply_time` | `bigint(20)` | `NULL DEFAULT NULL`，索引 `idx_last_apply_time` | 最后申请时间 |
| `status` | `tinyint(1)` | `NULL DEFAULT NULL` | 申请状态：`0` 待处理，`1` 已同意，`2` 已拒绝，`3` 已拉黑 |
| `apply_info` | `varchar(100)` | `NULL DEFAULT NULL` | 申请信息/验证消息 |

主键：`apply_id`

索引：`idx_key(apply_user_id, receive_user_id, contact_id)` 唯一索引，`idx_last_apply_time(last_apply_time)`

## user_info

表说明：用户信息表。

| 字段名 | 类型 | 约束/默认值 | 字段含义 |
| --- | --- | --- | --- |
| `user_id` | `varchar(12)` | `NOT NULL` | 用户 ID |
| `email` | `varchar(50)` | `NULL DEFAULT NULL`，唯一索引 `idx_key_email` | 邮箱 |
| `nick_name` | `varchar(20)` | `NULL DEFAULT NULL` | 昵称 |
| `join_type` | `tinyint(1)` | `NULL DEFAULT NULL` | 加好友方式：`0` 直接加入，`1` 同意后加好友 |
| `sex` | `tinyint(1)` | `NULL DEFAULT NULL` | 性别：`0` 女，`1` 男 |
| `password` | `varchar(32)` | `NULL DEFAULT NULL` | 密码 |
| `personal_signature` | `varchar(50)` | `NULL DEFAULT NULL` | 个性签名 |
| `status` | `tinyint(1)` | `NULL DEFAULT NULL` | 用户状态，具体枚举需以后端代码为准 |
| `create_time` | `datetime` | `NULL DEFAULT NULL` | 创建时间 |
| `last_login_time` | `datetime` | `NULL DEFAULT NULL` | 最后登录时间 |
| `area_name` | `varchar(50)` | `NULL DEFAULT NULL` | 地区 |
| `area_code` | `varchar(50)` | `NULL DEFAULT NULL` | 地区编号 |
| `last_off_time` | `bigint(13)` | `NULL DEFAULT NULL` | 最后离开时间 |

主键：`user_id`

索引：`idx_key_email(email)` 唯一索引

## user_info_beauty

表说明：靓号表。用于给指定邮箱分配或绑定特殊用户 ID。

| 字段名 | 类型 | 约束/默认值 | 字段含义 |
| --- | --- | --- | --- |
| `id` | `int(11)` | `NOT NULL AUTO_INCREMENT` | 自增 ID |
| `email` | `varchar(50)` | `NOT NULL`，唯一索引 `idx_key_email` | 邮箱 |
| `user_id` | `varchar(12)` | `NOT NULL`，唯一索引 `idx_key_user_id` | 用户 ID |
| `status` | `tinyint(1)` | `NULL DEFAULT 0` | 使用状态：`0` 未使用，`1` 已使用 |

主键：`id`

索引：`idx_key_user_id(user_id)` 唯一索引，`idx_key_email(email)` 唯一索引

## 聊天相关核心关系

| 关系 | 说明 |
| --- | --- |
| `chat_message.session_id` -> `chat_session.session_id` | 消息归属于某个会话 |
| `chat_session_user.session_id` -> `chat_session.session_id` | 用户和联系人之间关联到一个会话 |
| `chat_session_user.user_id` -> `user_info.user_id` | 会话所属用户 |
| `chat_session_user.contact_id` -> `user_info.user_id` 或 `group_info.group_id` | 单聊时关联用户，群聊时关联群组 |
| `user_contact.user_id` -> `user_info.user_id` | 联系关系所属用户 |
| `user_contact.contact_id` -> `user_info.user_id` 或 `group_info.group_id` | 好友 ID 或群组 ID |
